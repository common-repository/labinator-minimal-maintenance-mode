== Description ==

**Labinator Minimal Maintenance Mode** is a WordPress plugin that allows you to put your website in a coming soon or maintenance mode. It also allows you to set your own custom page for the content. It is designed to be fast and minimal without limitations or paid upgrades.

## Features

* Lightweight and minimal.
* Ability to set a maintenance mode for your website.
* Ability to set a coming soon page for your website.
* Ability to assign your own custom page as your maintenance or coming soon page instead of the default one.
* Ability to generate a private link to bypass the maintenance mode.
* 100% free and open-source without limitations or paid upgrades.

== Installation ==

### Method 1: From The WordPress Plugins Page
1. Go to your **`Plugins`** page.
2. Search for \"**Labinator Minimal Maintenance Mode**\" then click the **Enter** key.
3. Install the plugin then activate it.

### Method 2: By Uploading The ZIP File
1. Go to your **`Plugins`** page.
2. Upload the plugin zip file and install it.
3. Activate the plugin through the **`Plugins`** menu in WordPress.

### Method 3: Manually Using FTP
1. Upload the plugin folder to the **`/wp-content/plugins/`** directory.
2. Activate the plugin through the **`Plugins`** menu in WordPress.

== Frequently Asked Questions ==

= Will this bloat my website or slow it down? =
This plugin is designed to be extremely minimal and lightweight. It should not slow your website.

= Is it possible to customize the maintenance or coming soon page? =
The plugin provides an option where you can assign any custom page on your website to be your coming soon or under construction page. However, there are no options to customize the "default" page without editing the plugin source code files.

= Is there any upgrade or Pro version? =
This plugin is 100% free and open source. There are no upgrades or pro versions.
